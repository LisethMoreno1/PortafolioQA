# Portafolio de QA

Este repositorio contiene casos de prueba y proyectos destacados que demuestran mis habilidades en Quality Assurance.

## Casos de Prueba Ejemplares

En este repositorio encontrarás una colección de casos de prueba diseñados :

- **Test Case Manuales**: Pruebas manuales para la interfaz de usuario, con enfoque en la usabilidad.
- **Pruebas Actumatizadas**: Casos de prueba automatizados utilizando Selenium y Python.
- **Pruebas de Api**: Suite de pruebas de API utilizando Postman y JavaScript.
